CSS, relative vs absolute vs fixed positioning

To position elements on a page CSS offers many different ways

Position types in CSS

  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky

Relative positioning

Relative positioning adjusts the element respective of its normal position. When used alone (position:relative;) without other attributes nothing will happen and will look like "position: static;". When used with a position attribute you can move the element from its normal position e.g 70px from the top and 30px from the left from its original positions.

Absolute positioning

Absolute positioning allows you to select exactly where on the page you want the element. You can specify the exact top, left, bottom and right locations within the parent (e.g document body). However, the now removes the element from the flow of the page. This means this element does not affected and does not affect other elements.

Fixed positioning

Fixed positioning allows you to select exactly where in the viewport you want the element. You can specify the exact top, left, bottom and right locations relative to the viewport. However, this also now removes the element from the flow of the page. This means this element does not affected and does not affect other elements.
